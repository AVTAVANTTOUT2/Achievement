# Achievement — repoaudit

CLI Python léger pour auditer la qualité communautaire d'un dépôt GitHub local.

## Installation rapide

```bash
python3 -m pip install -e ".[dev]"
```

## Usage

```bash
python3 -m repoaudit .
repoaudit --strict .
```

## Tests

```bash
python3 -m pytest -q
```
